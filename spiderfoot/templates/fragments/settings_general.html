General

Core operational settings for SpiderFoot.

{# Max Threads #}

Higher values speed up scans but use more CPU and memory. Start with 3 and increase if your system can handle it.

{# DNS Server #}

Override the system DNS resolver. Use 8.8.8.8 (Google) or 1.1.1.1 (Cloudflare) for reliability.

{# HTTP User Agent #}

The browser identity string sent with HTTP requests. Changing this can help avoid being blocked by some websites.

{# Fetch Timeout #}

How long to wait for each HTTP response before giving up. Increase for slow or distant targets.
